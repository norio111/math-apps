# 🧮 Math Learning Apps

微分・数学の概念をインタラクティブに学ぶためのWebアプリ集です。
授業で習った内容を手を動かしながら理解するために作りました。

## 📚 アプリ一覧

| ファイル名 | 内容 |
|---|---|
| binomial_theorem.html | 二項定理の学習アプリ |
| calculus_learning_app.html | 微分マスター |
| calculus_ml_app.html | 変化率と傾き |
| chain_rule_free_input.html | 式を入力して連鎖律を見る |
| math_drill.html | 微分ドリル |

## 🚀 使い方

各HTMLファイルをブラウザで開くだけで使えます。インストール不要！

## 🛠️ 使用技術

- HTML / CSS / JavaScript

## 📝 作成背景

AIエンジニアを目指す学習の一環として、
機械学習に必要な数学（微分・連鎖律など）を
自分で手を動かしながら理解するために作成しました。
Claude（AI）と一緒に作っています。
